# 📊 Data Analysis & Business Insights Project

## 🔍 Overview
This project focuses on analyzing a sales dataset to extract meaningful business insights.  
It includes data cleaning, profitability analysis, Pareto analysis, and performance evaluation at product and division levels.

---

## 🎯 Objectives
- Clean and preprocess raw data  
- Calculate key business metrics  
- Identify high-performing products  
- Detect cost inefficiencies  
- Support data-driven decision making  

---

## ⚙️ Technologies Used
- Python 🐍  
- Pandas  
- Matplotlib  
- VS Code  

---

## 📁 Project Workflow

### 1️⃣ Data Cleaning
- Removed missing and invalid values  
- Handled duplicates  
- Standardized data  

### 2️⃣ Profitability Analysis
- Calculated:
  - Total Profit  
  - Profit per Unit  
  - Gross Margin (%)  

### 3️⃣ Pareto Analysis (80/20 Rule)
- Identified top products contributing to majority of profit  
- Analyzed revenue and profit concentration  

### 4️⃣ Product-Level Analysis
- Ranked products based on profit  
- Identified:
  - High-performing products  
  - Low-performing products  

### 5️⃣ Division-Level Analysis
- Compared performance across divisions  
- Evaluated revenue vs profit contribution  

### 6️⃣ Cost Structure Diagnostics
- Identified cost-heavy products  
- Detected pricing inefficiencies  
- Suggested optimization strategies  

---

## 📊 Output Files
- `cleaned_data.csv`  
- `final_profit_data.csv`  
- `product_analysis.csv`  
- `division_analysis.csv`  
- `cost_issues_products.csv`  

---

## 🚀 Key Insights
- A small percentage of products generate the majority of profit (Pareto Principle)  
- Some products have high sales but low profit margins  
- Certain divisions perform significantly better than others  
- Cost inefficiencies were identified in multiple products  

---

## 🎯 Conclusion
This project demonstrates how data analysis can be used to improve business performance by identifying profitable products, reducing costs, and optimizing strategies.
